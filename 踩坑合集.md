# 踩坑合集
## 一、服务器相关
### 1. ssh免密连接
修改文件权限，```chmod 700 .ssh```。
### 2. 科学上网
- 检查是否登录联网，不同终端可能需要再次登录，仅ping通不代表连接外网，注意查看是否正常联网
```bash
curl -L www.baidu.com
```
- 使用镜像网站，如huggingface可通过hf-mirror下载模型，
```bash
HF_ENDPOINT=https://hf-mirror.com python your_script.py
```
- 通过clash设置代理
```bash
# 开启代理
~/.local/bin/clash
proxy
# 关闭代理
noproxy
```
